# 珠玑妙算 (Mastermind Game)

一个基于HTML/CSS/JavaScript的密码破译游戏，灵感来自经典的Mastermind桌游。

## 🎮 游戏介绍

珠玑妙算是一款益智解谜游戏，玩家需要在有限次数内猜测出由4个颜色组成的密码。

### 游戏规则

1. **游戏目标**：在10次尝试内猜测出由4个颜色组成的密码
2. **颜色选择**：从6种颜色中选择（红、蓝、绿、黄、紫、橙）
3. **重要规则**：密码中每个颜色只出现一次，不重复
4. **反馈机制**：
   - ⚫ **黑色**：颜色和位置都正确
   - ⚪ **白色**：颜色正确但位置错误
   - 🔘 **灰色**：颜色和位置都不对
5. **游戏结束**：猜对密码或用完10次尝试

### 操作方式

- **鼠标操作**：点击颜色按钮选择颜色，点击"提交猜测"按钮提交
- **键盘操作**：
  - 数字键1-6：选择对应颜色
  - 回车键：提交猜测
  - R键：重新开始游戏
  - 退格键：删除最后一个选择的颜色

## 🎉 特色功能

- 🎊 **胜利庆祝**：猜对密码时会有彩花、音效和动画效果
- 🎨 **精美界面**：现代化的深色主题设计
- ⌨️ **键盘支持**：支持键盘快捷键操作
- 📱 **响应式布局**：适配不同屏幕尺寸
- 🎯 **直观反馈**：猜测结果与反馈一一对应，易于理解

## 🛠️ 本地运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 克隆项目
git clone https://github.com/你的用户名/珠玑妙算.git

# 进入项目目录
cd 珠玑妙算

# 打开游戏
start index.html
```
